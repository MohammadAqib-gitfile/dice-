🎲 Dice Rolling Simulator (Python)

This is a simple Python-based dice rolling simulator that mimics the experience of rolling two dice. The program uses random number generation and ASCII art to visually display the dice faces in the terminal.

🚀 Features
Simulates rolling two dice simultaneously
Generates random values between 1 and 6
Displays dice visually using ASCII art
Allows users to roll repeatedly until they choose to stop
Beginner-friendly and easy to understand
🛠️ Technologies Used
Python
random module
📌 How It Works
The user is prompted to roll the dice.
If the user enters "Yes", the program:
Generates two random numbers (1–6)
Displays the corresponding dice faces
The user can continue rolling or exit the program.
▶️ How to Run
Make sure Python is installed on your system.
Copy the code into a .py file (e.g., dice_simulator.py)

Run the file using:

python dice_simulator.py
💡 Example Output
dice rolled: 3 and 5
